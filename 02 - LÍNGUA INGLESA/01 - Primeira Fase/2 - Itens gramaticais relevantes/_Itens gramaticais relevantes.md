## 1. For-Infinitive Clauses

### Core Structure

```
for + NP (logical subject) + to + verb (infinitive predicate)
```

### Syntactic Function

The **entire clause** functions as an **adverbial of purpose**, modifying the main verb.

> ⚠️ **Exam trap:** Never isolate "to + verb" alone and classify it as the adverb. The "for + NP" is inseparable from the infinitive — together they form a single syntactic unit.

### When to Use Each Structure

|Structure|When to use|
|---|---|
|**to + verb**|Subject is the **same** as the main clause|
|**for + NP + to + verb**|Subject is **different** from the main clause|

### Key Examples

- _"We never connected the dots **for women themselves to build the constituencies**."_ → "women themselves" = logical subject; "to build" = infinitive predicate; whole clause = adverbial of purpose modifying "connected"
- _"The law was changed **for women to have equal rights** in the workplace."_
- _"The door was left open **for the children to come in** freely."_

### Contrast to Memorize

> _"I studied hard **to pass the exam**."_ → same subject (I) → simple infinitive _"I stayed up **for my friend to pass the exam**."_ → different subject (my friend) → for-infinitive clause

## 2. Passive Voice in English

### The Golden Rule

> If the **subject receives, suffers, or experiences** the action — and you can identify or imagine an **external agent** — you are dealing with passive voice.

### Standard Formula

```
BE (any tense) + past participle
```

### Passive Voice Across All Tenses

|Tense|Example|
|---|---|
|Simple Present|_Women **are excluded** from decisions._|
|Simple Past|_Women **were excluded** from decisions._|
|Present Perfect|_Women **have been excluded** from decisions._|
|Past Perfect|_Women **had been excluded** from decisions._|
|Future|_Women **will be excluded** from decisions._|
|Modal|_Women **should be included** at the table._|
|Present Continuous|_Women **are being excluded** right now._|

### Beyond Standard BE — Other Passive Markers

#### GET Passive (informal register)

> _Women **got ignored** at the meeting._

#### Stative Passive — linking verb + past participle

The subject is in a **state resulting from** a passive action.

|Linking Verb|Example|
|---|---|
|**remain**|_Women **remain relegated** to the margins._|
|**seem**|_The problem **seems overlooked** by policymakers._|
|**appear**|_The issue **appears unresolved** after negotiations._|
|**stay**|_The door **stayed closed** throughout the session._|
|**become**|_She **became respected** worldwide._|
|**feel**|_He **felt defeated** after the vote._|

> 💡 **"remain relegated"** (from the CACD text) = **stative passive**: "remain" is a linking verb; "relegated" is a participial adjective carrying passive meaning. The subject (women) did not relegate themselves — an external systemic force did.

### The Three-Question Test

1. **Is there a past participle?** → If NO, almost certainly not passive.
2. **Is the subject receiving the action?** → If NO, may be active perfect tense.
3. **Could you add "by someone/something" naturally?** → If YES, it is passive.

### Classic Trap — Passive vs. Pure Adjective

|Sentence|Classification|Reason|
|---|---|---|
|_The window **was broken** by vandals._|✅ Passive|Agent explicit|
|_The window **was broken** when I arrived._|⚠️ Ambiguous|Could be passive or stative|
|_The window **looked broken**._|❌ Not passive|Pure adjective; no agent possible|

> 🔑 **The "by" test:** Can you add "by + agent" and keep it natural? If yes → passive. If no → adjective.

### Dynamic vs. Stative Passive

|Type|Example|Emphasis|
|---|---|---|
|**Dynamic passive**|_Women **are being relegated**._|Action in progress|
|**Stative passive**|_Women **remain relegated**._|Resulting, ongoing state|

## 3. Present Participle vs. Verb Tenses — The -ING Trap

### The Core Problem

The **-ing ending** appears in multiple different grammatical structures. Do NOT classify a structure by the -ing ending alone — always check for **auxiliary verbs**.

### The Four -ING Structures

|Structure|Formula|Example|Function|
|---|---|---|---|
|**Present Continuous**|IS/ARE + -ing|_She **is adopting** new tools._|Ongoing action now|
|**Present Perfect Continuous**|HAVE/HAS + BEEN + -ing|_She **has been adopting** tools._|Action from past continuing now|
|**Participial Phrase**|-ing alone (no auxiliary)|_**Adopting** new tools, she improved results._|Modifies the subject|
|**Reduced Relative Clause**|NP + -ing|_Institutions **adopting** new tools..._|Identifies which noun|

> ⚠️ **CACD Exam Case:** _"Diplomatic institutions **adopting** such technologies should also adopt guardrails..."_ → NO auxiliary verb present → NOT present perfect continuous → "adopting" = **present participle** forming a **reduced relative clause** → Equivalent to: _"Diplomatic institutions **that adopt** such technologies..."_

### Present Perfect Continuous — Mandatory Formula

```
HAVE / HAS + BEEN + verb-ING
```

> Without **both** auxiliaries (_have/has_ AND _been_), the structure **cannot** be present perfect continuous.

### Reduced Relative Clause — Key Facts

- Formed by removing the relative pronoun + auxiliary BE
- Extremely common in **academic and formal writing**
- Functions as a **noun modifier** (not a tense)
- Has **no time reference** of its own — takes its time from the main clause verb

|Full Relative Clause|Reduced Form|
|---|---|
|_institutions **that adopt** technologies_|_institutions **adopting** technologies_|
|_diplomats **who are negotiating** the treaty_|_diplomats **negotiating** the treaty_|
|_countries **that are developing** AI_|_countries **developing** AI_|

## 4. Quick-Reference: Key Distinctions

|Structure|Key Marker|Do NOT confuse with|
|---|---|---|
|**For-infinitive clause**|for + NP + to + V|Simple infinitive of purpose (no "for + NP")|
|**Standard passive**|BE + past participle|Active perfect (HAVE + past participle)|
|**Stative passive**|Linking verb + past participle|Pure adjectival construction|
|**Present perfect continuous**|HAVE/HAS + BEEN + -ing|Present participle / reduced relative clause|
|**Reduced relative clause**|NP + -ing (no auxiliary)|Present continuous (needs IS/ARE)|
